## Project Overview

**bridgeUI** is a real-time UI translation plugin aimed at resolving digital symbol (icon) barriers for elderly users with low digital literacy ("디지털 기호(Icon) 장벽 해소를 위한 실시간 UI 번역 플러그인 구현 - 노년층 디지털 리터러시").

The project is currently in initialization phase — no source code or tooling has been added yet.
